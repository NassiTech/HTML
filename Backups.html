Backups

What is a backup?

A backup is a copy of important data (files, databases, system settings, etc.) that is stored separately to protect against data loss. If the original data is lost, damaged, or corrupted, the backup can be used to restore it.

Why should we have backups?

Backups are maint to protect the user against:

  1. cyber attaks: hacking, viruses, ransomware
  2. software issues
  3. accidental deletion
  4. hardware failure
  5. natural disasters
